TodoList 앱 디버깅

현재 서버: http://localhost:4003

확인사항:

  1. 위 링크를 클릭해서 앱이 열리는지 확인
  2. 개발자 도구(F12) → Console 탭에서 오류 확인
  3. 화면에 할일 목록이 표시되지 않으면 구글 로그인 시도
  4. 여전히 비어있으면 콘솔에서 다음 명령어 실행:

콘솔 명령어:

// 현재 할일 개수 확인
console.log('현재 할일 개수:', document.querySelectorAll('[data-todo-id]').length)

// 로그인 상태 확인  
console.log('로그인된 사용자:', localStorage.getItem('firebase:authUser:AIzaSyD8aBPw-o13mciSdV8gzRJN6TfaAy3OoWg:[DEFAULT]'))

// Firestore 연결 상태 확인
if (window.firebase) {
    console.log('Firebase 초기화됨')
} else {
    console.log('Firebase 초기화 안됨')
}

이 결과를 알려주시면 정확한 문제를 찾아서 해결하겠습니다!
